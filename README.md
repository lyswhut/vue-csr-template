# vue-csr-template

A vue-CSR based scaffolding.

Scaffolding mainly integrates the following packages:

* vue 3.x
* vue-router
* vuex
* axios
* webpack 5.x
* babel 7.x
* autoprefixer
* eslint
* less
* stylus
* pug

## How to use

```bash
# dev mode
npm run dev

# production build
npm run build

# eslint check
npm run lint

# eslint fix
npm run lint-fix

```
